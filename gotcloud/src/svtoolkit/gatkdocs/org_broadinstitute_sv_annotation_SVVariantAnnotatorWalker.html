SVVariantAnnotatorWalker

Walker interface that implements SVAnnotator.

Category Variant Annotation


Introduction

This walker is not normally used directly, but is called through the SVAnnotator command-line interface. This walker documents the union of all arguments for all SV Annotators.

See the SVAnnotator documentation for a description of the annotator framework and the arguments common to all annotators.

See the documentation for each specific variant annotators for documentation on the arguments specific to each individual annotator.
